What is that?

HTML is a pretty simple programming language made for basically marking up a web page. It means it's a page's carcass which contains text and sets page's structure

Who invented HTML?

HTML was developed by Tim Berners-Lee, an Engish physicist, in 1990.

What are basic HTML elements?

  • <p></p>-element which represents a paragraph
  • <h1></h1>-element for the main heading. Commonly used once a page.
  • <head></head>-"head" of the document. It doesn't display on page in any way but it contains a title of the page(in browser) and some metadata.
  • <body></body>-contains all of the visible information
  • <footer></footer>-always located in the end of the page. Commonly contains author info and contacts.
